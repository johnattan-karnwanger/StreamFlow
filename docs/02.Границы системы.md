#### Границы системы

| Внутренние   | Внешние интегрированные         |
| ------------ | ------------------------------- |
| Frontend     | CDN                             |
| Backend      | Платежная система для подписок  |
| БД           | Стриминговое ПО (OBS и аналоги) |
| Media Server | SMS-шлюз                        |

#### Пояснение

- Пользователь взаимодействует с системой через **Frontend**. 
- Frontend обращается к **Backend**, который управляет бизнес-логикой: проведением трансляций, подписками, модерацией и рекомендациями. 
- Backend при необходимости читает и записывает данные в **БД**. 
- Видеопоток доставляется зрителям через **CDN** — внешнюю сеть доставки контента. 
- **Платёжная система** обрабатывает оплату подписок через Backend.
