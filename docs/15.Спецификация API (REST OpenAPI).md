#### Эндпоинты

**Аутентификация**
1. POST /auth/register
2. POST /auth/verify-code
3. POST /auth/login

**Пользователи**
1. GET /users/{id}
2. GET /users/{id}/role
3. PATCH /users/{id}
4. DELETE /users/{id}

**Каналы**
1. POST /channels
2. GET /channels
3. GET /channels/{id}
4. PATCH /channels/{id}
5. DELETE /channels/{id}
6. POST /channels/{id}/followers
7. DELETE /channels/{id}/followers

**Стримы**
1. POST /streams
2. GET /streams/{id}
3. PATCH /streams/{id}
4. DELETE /streams/{id}
5. GET /streams/{id}/stream-key
6. POST /streams/{id}/messages
7. GET /streams/{id}/messages
8. DELETE /streams/{id}/messages/{message_id}

**VOD**
1. GET /vods/{id}
2. PATCH /vods/{id}
3. DELETE /vods/{id}

**Подписки**
1. POST /subscriptions
2. GET /subscriptions/{id}
3. PATCH /subscriptions/{id}
4. DELETE /subscriptions/{id}

**Модерация**
1. POST /moderation
2. GET /moderation
3. GET /moderation/{id}
4. PATCH /moderation/{id}
5. DELETE /moderation/{id}

**Жалобы**
1. POST /reports
2. GET /reports
3. GET /reports/{id}
4. PATCH /reports/{id}
5. DELETE /reports/{id}
